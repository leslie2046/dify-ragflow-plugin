# Privacy Policy

## Data Handling
This plugin serves as a connector between Dify and RAGFlow.
- **Data Transmission**: User inputs (such as queries and configuration details) are transmitted to the RAGFlow instance specified by the user in the plugin configuration.
- **Data Storage**: This plugin does not persistently store any user data or processed content. All data management is handled by the connected RAGFlow instance.

## Third-Party Services
- **RAGFlow**: This plugin interacts with the RAGFlow API. Users are responsible for ensuring their RAGFlow instance complies with their data privacy requirements.

## Changes
We may update this privacy policy to reflect changes in the plugin's functionality.
